# Music_hnust
校srip项目，将科大的校园地图与音乐软件结合起来。项目包含客户端和服务端，客户端通过学号进行登录，用户可以发布动态，查看动态，评论，创建本地歌单，分享歌单内容，地图定位，查询科大地图，本地音乐播放，在线听歌，下载歌曲，摇一摇搜索听歌的人等功能。
